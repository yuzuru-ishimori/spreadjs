コピー&ペースト 開発予定

範囲選択してコピー・貼り付け。型変換・複数セルの一括貼り付け・競合検出・元に戻す（Undo）まで一貫して扱います。

Stage 2 先頭

行の挿入・削除 開発予定

共同編集中でも整合が崩れない行操作。複数人が同時に行を挿入・削除してもデータは正しく収束します。

Stage 2

数式 開発予定

四則演算・セル参照・SUM。行を移動しても参照が壊れない方式で、依存セルは自動再計算されます（技術検証は完了済み）。

Stage 2

配布・運用の本格化 開発予定

社内パッケージレジストリからの配布、ビルド済み配布物、PostgreSQL による永続化、React 用ラッパー、テレメトリ。

Stage 2

対象外 設計上の境界

「まだ」ではなく、品質を集中させるために意図的に対象外と決めている範囲です（Alpha 製品境界として文書化済み）。

macOS・Firefox・モバイル

対応環境を Windows の Chrome / Edge に集中して品質を担保します。対応拡大は将来段階で判断します。

本番運用機能

認証認可・テナント分離・高可用性・バックアップ運用は SDK の範囲外。信頼できる社内環境での利用を前提とし、運用要件は利用側システムが担います。

オフライン編集

長時間オフラインでの編集継続や複数端末のオフライン統合は保証しません（一時的な切断は「切断・再接続耐性」でカバーします）。

長期のAPI後方互換

Alpha 期間の API は 0.x（実験的）。変更はすべて変更履歴で追跡できますが、長期の後方互換は約束しません。
